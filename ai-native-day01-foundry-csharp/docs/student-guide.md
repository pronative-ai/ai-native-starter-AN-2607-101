# Day 1 Student Guide

## What You Build

You build a small C# console app that calls a live Foundry/Azure OpenAI model deployment and optionally grounds the answer with Azure AI Search.

## Walkthrough

1. Confirm you can access the Foundry project in the portal.
2. Compare models and note which deployment the trainer assigns.
3. Run the C# app with a simple prompt.
4. Add grounding by enabling Azure AI Search settings.
5. Review console traces and Foundry run visibility.
6. Trigger a responsible AI/content filter scenario from the trainer walkthrough.

## Key Learning

The goal is not to memorize SDK calls. The goal is to understand how an AI-native app connects:

- model endpoint
- grounding source
- identity
- telemetry
- safety behavior
