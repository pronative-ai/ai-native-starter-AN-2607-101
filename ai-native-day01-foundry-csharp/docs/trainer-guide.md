# Day 1 Trainer Guide

## Trainer Flow

Use the MS Learn AI-3016 exercise sequence as the authority path. Use this C# repo when the exercise needs a ProNative C# live-environment companion.

## Suggested Timing

| Segment | Activity |
|---|---|
| Foundry orientation | Portal walkthrough using live batch resources |
| Model catalog/evaluation | Compare assigned deployments |
| C# app | Run `Day01FoundryChat` and inspect request/response |
| RAG | Show Search index naming and grounding behavior |
| Responsible AI | Demonstrate content filtering and discuss traceability |

## Trainer Notes

- Keep the code walkthrough slow and explicit.
- Ask students to identify which Azure resource each setting maps to.
- Use standard Foundry visibility and Application Insights/Azure Monitor only.
- Save ProNative custom workbooks for Days 6 to 8.
