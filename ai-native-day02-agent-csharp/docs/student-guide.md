# Day 2 Student Guide

## What You Build

You run a small C# agent API that accepts a user message, decides whether a tool is needed, calls a custom tool, and returns an answer.

## Walkthrough

1. Build an agent in Foundry following the trainer/MS Learn flow.
2. Run this local C# companion API.
3. Call `/agent/run` with a simple question.
4. Call `/agent/run` with an order-status question to trigger a tool.
5. Inspect `/.well-known/mcp/tools`.
6. Review where Foundry IQ would provide knowledge grounding.
7. Observe console traces and Foundry run visibility.
