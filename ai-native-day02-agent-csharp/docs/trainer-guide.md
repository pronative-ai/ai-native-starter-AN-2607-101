# Day 2 Trainer Guide

## Trainer Flow

Use the MS Learn AI-3026 exercise sequence as the authority path. Use this repo to show the C# shape of an agent runtime and tool integration.

## Suggested Timing

| Segment | Activity |
|---|---|
| Agent lifecycle | Foundry portal/VS Code walkthrough |
| Custom tools | Run `/tools/order-status`, then trigger via `/agent/run` |
| MCP | Show discovery-style metadata endpoint |
| Foundry IQ | Trainer demo or controlled walkthrough |
| M365/Teams | Trainer-demo only |
| Workflows/multi-agent | Preview only; defer depth to Days 3 and 4 |

## Trainer Notes

- Keep Day 2 centered on single-agent lifecycle.
- Do not spend deep implementation time on multi-agent orchestration.
- Show observability using standard console traces, Foundry visibility, and App Insights/Azure Monitor if configured.
