# Student Support Triage

Use this skill during practical delivery when a student is blocked.

Follow this process:

1. Read `references/support-flow.md`.
2. Identify whether the issue is local device, repository, identity, Azure RBAC, Foundry, or service quota.
3. Ask for the smallest missing evidence.
4. Give one concrete next action at a time.

Avoid broad troubleshooting checklists unless the student has not provided any symptom.
