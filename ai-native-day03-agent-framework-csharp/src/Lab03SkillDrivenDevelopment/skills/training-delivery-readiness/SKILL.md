# Training Delivery Readiness

Use this skill before a live practical session or when a trainer asks whether the batch is ready.

Follow this process:

1. Read `references/readiness-checklist.md`.
2. Check whether shared resources, student-specific resources, and repositories are ready.
3. Report missing evidence as action items.
4. Do not say the batch is ready unless every critical dependency has evidence.

Return:

- readiness status
- missing evidence
- operational risk
- next action

Keep the response concise and practical.
